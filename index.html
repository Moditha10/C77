Project on console

It's a streamlined code editor with debugging, task running, and version control features. It provides just the tools a developer needs for quick code-build-debug cycles.

A console is a text-based interface that allows a user to interact with a computer program. It typically displays text output from the program and accepts text input from the user. Consoles are commonly used in command-line interface (CLI) programs and in some integrated development environments (IDEs).
